## SECTION 2: GLOBAL DESIGN SPECIFICATIONS

### Color Palette
- **Primary Background:** #000000 (pure black)
- **Secondary Background:** #0A0A0F (dark charcoal for cards)
- **Primary Accent:** #10B981 (emerald green — used for key numbers, CTAs, highlights)
- **Secondary Accent:** #3B82F6 (blue — used for secondary data, environmental themes)
- **Tertiary Accent:** #8B5CF6 (purple — used for tertiary elements, legal/justice themes)
- **Alert/Problem Accent:** #EF4444 (red — used only for problem slides, health stats)
- **Text Primary:** #FFFFFF (white)
- **Text Secondary:** #9CA3AF (gray-400)
- **Text Muted:** #6B7280 (gray-500)
- **Card Fill:** rgba(255, 255, 255, 0.03) (3% white)
- **Card Border:** rgba(255, 255, 255, 0.06) (6% white)

### Typography
- **Headlines:** Bold, 36-48pt, white, tight letter-spacing (-0.02em)
- **Subheadlines:** Semi-bold, 20-24pt, gray-400 or emerald
- **Body Text:** Regular, 14-18pt, gray-300
- **Stats/Numbers:** Extra-bold, 48-72pt, emerald/blue/purple accents
- **Sources/Citations:** Light, 10-12pt, gray-600
- **Font Family:** Inter, SF Pro Display, or any clean sans-serif. Avoid serif fonts entirely. Avoid decorative fonts. The typography should feel modern, technical, and trustworthy.

### Layout Principles
- **Whitespace is mandatory.** Every slide should have at least 15-20% empty space. Do not fill every pixel.
- **One idea per slide.** No slide should try to communicate more than one core concept.
- **Glass-morphism cards** for data containers — semi-transparent backgrounds with backdrop blur, thin borders
- **Consistent margins:** 60-80px from all edges on every slide
- **No clip art.** No WordArt. No gradient text (except for the specific "gradient-text" treatment on key phrases). No 3D effects. No transition animations between slides (simple fade only if needed).
- **Slide aspect ratio:** 16:9 (widescreen)
- **Maximum 6-7 bullet points per slide.** If you have more, split into two slides.

### Image Guidelines
- Use high-quality, high-resolution images only (minimum 1920x1080)
- All factory farm images should show the *industrial scale* — aerial shots, long rows of barns, waste lagoons, not individual animals in close-up
- Apply dark overlays (60-80% black) on all background images so text remains readable
- App screenshots should be actual mockups or screenshots of the AFFOG interface with the dark theme
- Icons should be from Lucide (the icon library used in the app) or a similar minimal line-icon set
